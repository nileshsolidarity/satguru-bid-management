How to Sync Portals

The Sync Portals feature uses a browser automation script (Playwright) to log into Global Tenders & Tenders Info and pull live tenders. This requires running the backend on your Mac — it cannot run on Vercel's servers.

To sync tenders, open Terminal and run:
cd "/Users/satguru/Bid Management tool"
python3 server.py
Then open http://localhost:3000 — the Sync button will work there.
